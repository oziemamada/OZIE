

                         Ultimate Google Analytics
                         =========================

This is a WordPress plugin to add JavaScript to your weblog for using 
Google Analytics. It does not only add the basic JavaScript to track your
pages, but can also add JavaScript to outbound, mailto: and download links
so you can also track clicks on these links.

For more information, downloads, version history and installation intructions
please visit http://www.oratransplant.nl/uga
